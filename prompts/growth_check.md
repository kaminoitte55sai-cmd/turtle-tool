# 成長性チェック
# 対象銘柄: {{TARGET_CODE}}

EdinetDB を使って {{TARGET_CODE}} の成長性を分析してください。

## 手順
1. `search_companies` で EDINETコードを取得
2. `get_financials` で過去5年の財務時系列を取得（years=5）
3. `get_company` で直近データと latestEarnings を取得
4. `get_segments` でセグメント別成長を確認
5. `get_text_blocks` で「経営者による分析」セクションを取得

## 出力

### 成長指標サマリー
| 指標 | 3年CAGR | 5年CAGR | 直近YoY |
|------|---------|---------|---------|
| 売上高 | | | |
| 営業利益 | | | |
| EPS | | | |
| ROE | | | |

### 成長の質
- **売上成長の中身**：価格上昇 / 数量増加 / M&A寄与 を有報から判定
- **利益率トレンド**：営業利益率の推移（改善/横ばい/悪化）
- **EPS成長の質**：自社株買い効果を除いた実力EPS成長率の推定

### セグメント別成長（取得できた場合）
- 成長セグメント・縮小セグメントの特定

### 今後の成長ドライバー（経営者コメントより）
- 会社が重点投資している領域
- 数値目標（中期経営計画があれば）

### 成長株としての評価
- PEG レシオ（PER ÷ EPS成長率）の計算と評価（1.0以下なら割安成長株）
- **高成長** / **安定成長** / **成熟** / **縮小懸念** の判定と根拠

出力は日本語・数値を重視して。
