# 事業リスク・競合分析
# 対象銘柄: {{TARGET_CODE}}

EdinetDB を使って {{TARGET_CODE}} のリスクと競合状況を分析してください。

## 手順
1. `search_companies` で EDINETコードを取得
2. `get_text_blocks` で有報のリスク・事業状況セクションを取得
3. `get_company` で財務健全性スコアと財務データを取得
4. `screen_companies` で同業種の競合企業を抽出（ROE降順 10社）
5. `search_companies_batch` で競合の主要指標を一括取得

## 出力

### 事業概要（3行以内）

### 主要リスク（有報記載より上位5件）
各リスクについて：
- **リスク名**：内容の要約（2行）、重要度（高/中/低）

### 財務面のリスク
- 自己資本比率・D/Eレシオ・FCFから見た財務健全性
- 健全性スコア X/100 の評価コメント

### 競合比較（上位5社）

| 企業名 | ROE | 営業利益率 | PER | 健全性 |
|--------|-----|----------|-----|--------|
| **{{TARGET_CODE}}（本銘柄）** | | | | |
| 競合① | | | | |
| ... | | | | |

### 競合優位性の評価
- 強み：競合と比較した優位点
- 弱み：競合に劣る点
- 総合評価：業界内ポジション（上位/中位/下位）

出力は日本語・箇条書き中心で。
