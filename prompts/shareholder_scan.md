# 株主構成スキャンプロンプト
# 使い方: claude -p "$(cat prompts/shareholder_scan.md)" --allowedTools edinetdb

## タスク
EdinetDB MCP (`edinetdb`) を使って、指定銘柄の株主構成を詳細分析してください。

## 分析対象
- **銘柄コード**: {{TARGET_CODE}}  ← 実行前に証券コードに書き換えてください

## 手順

### Step 1: 現在の大株主情報
- `get_shareholders` で主要株主（上位10名）を取得
- 氏名・保有株数・保有比率を一覧化

### Step 2: 株主構成の時系列変化
- `get_shareholder_history` で過去3年の株主変化を取得
- 新規参入・撤退した主要投資家を特定

### Step 3: アクティビスト確認
- `get_activist_positions` でアクティビスト投資家のポジションを確認
- 存在する場合は要求内容・保有比率・進捗を整理

### Step 4: 政策保有株（持ち合い）の確認
- `get_cross_shareholdings` で政策保有株情報を取得
- コーポレートガバナンスへの影響を評価

## 出力フォーマット

### 1. 大株主トップ10

| 順位 | 株主名 | 保有比率 | 種別 |
|------|--------|---------|------|
| 1 | | | 金融機関/事業会社/外国法人/個人 |
| ... | | | |

### 2. 株主構成サマリー
- 安定株主比率（創業家＋持ち合い＋従業員持株）: XX%
- 機関投資家比率: XX%
- 外国人持株比率: XX%

### 3. 直近1年の変化
- 増加した主要投資家と変化量
- 減少した主要投資家と変化量

### 4. アクティビスト情報
- 存在する場合: 投資家名・保有比率・主な要求内容
- 存在しない場合: 「アクティビスト保有なし」

### 5. 政策保有株の評価
- 政策保有株の規模感
- ガバナンス上のリスク評価（高/中/低）

### 6. 投資判断への影響
- 株主構成から見た安定性・変動リスクの総評
