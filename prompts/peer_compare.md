# 同業他社比較分析プロンプト
# 使い方: claude -p "$(cat prompts/peer_compare.md)" --allowedTools edinetdb

## タスク
EdinetDB MCP (`edinetdb`) を使って、指定銘柄と同業他社を比較分析してください。

## 分析対象
- **メイン銘柄**: {{TARGET_CODE}}  ← 実行前に証券コードに書き換えてください
- **比較対象**: 同業種の主要企業（`screen_companies` で自動抽出）

## 手順

### Step 1: メイン銘柄の情報取得
- `get_company` でメイン銘柄の基本情報・財務ハイライトを取得
- `get_financials` で過去3年の財務時系列を取得

### Step 2: 同業他社の抽出
- `search_companies` または `screen_companies` でメイン銘柄と同業種の企業を10社抽出
- 条件: 同じ業種 (`industry`)、時価総額が近い企業を優先

### Step 3: 一括比較データ収集
- `search_companies_batch` で抽出した銘柄を一括取得

### Step 4: 比較表の作成
以下の指標を横並びで比較してください：

| 指標 | メイン銘柄 | 他社① | 他社② | 他社③ | 業界平均 |
|------|-----------|-------|-------|-------|---------|
| 売上高 (億円) | | | | | |
| 売上高成長率 (YoY) | | | | | |
| 営業利益率 (%) | | | | | |
| ROE (%) | | | | | |
| ROA (%) | | | | | |
| 自己資本比率 (%) | | | | | |
| PER (倍) | | | | | |
| PBR (倍) | | | | | |
| 配当利回り (%) | | | | | |
| 財務健全性スコア | | | | | |

### Step 5: 総合評価
- メイン銘柄の**強み・弱み**を箇条書きで整理
- 同業他社と比較した**割安・割高**の判定
- **投資判断に関わるリスク**を列挙（`get_text_blocks` のリスク情報を参照）

## 出力フォーマット
- 日本語で出力
- 数値は見やすく単位付きで表記（例: 1,234億円、15.3%）
- 最後に「まとめ」として3行以内の結論を記載
