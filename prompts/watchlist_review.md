# 監視銘柄レビュープロンプト
# 使い方: claude -p "$(cat prompts/watchlist_review.md)" --allowedTools edinetdb

## タスク
EdinetDB MCP (`edinetdb`) のウォッチリストに登録されている全銘柄を一括レビューしてください。

## 手順

### Step 1: ウォッチリスト取得
- `get_watchlist` で登録銘柄を全件取得

### Step 2: 全銘柄の一括データ取得
- `search_companies_batch` でウォッチリスト全銘柄の最新データを一括取得

### Step 3: 決算カレンダーの確認
- `get_earnings_calendar` で直近・今後の決算発表日を確認
- ウォッチリスト銘柄の決算予定を抽出

### Step 4: 直近決算がある銘柄の個別確認
- 直近30日以内に決算発表があった銘柄について `get_earnings` で内容確認
- 前回レビュー時からの変化点を特定

## 出力フォーマット

### 1. ウォッチリスト銘柄一覧（最新状態）

| 証券コード | 企業名 | 現在価格 | ROE | PER | 健全性スコア | ラベル |
|-----------|--------|---------|-----|-----|------------|--------|
| | | | | | | |

### 2. 注目アラート
以下の変化があった銘柄を強調表示:
- 🟢 **ポジティブ**: 業績上振れ・スコア改善・大株主増加
- 🔴 **ネガティブ**: 業績下振れ・スコア悪化・アクティビスト登場
- 🟡 **中立変化**: 決算発表予定・株主変動

### 3. 今後の決算スケジュール

| 日付 | 証券コード | 企業名 | 決算種別 |
|------|-----------|--------|---------|
| | | | 本決算/第N四半期 |

### 4. 個別コメント
変化があった銘柄について各2〜3行のコメント

### 5. 全体まとめ
- ウォッチリスト全体の状況評価（1〜2行）
- 今週/今月の優先確認銘柄の推奨
