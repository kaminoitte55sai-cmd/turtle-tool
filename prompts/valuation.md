# バリュエーション評価
# 対象銘柄: {{TARGET_CODE}}

EdinetDB を使って {{TARGET_CODE}} のバリュエーションを多角的に評価してください。

## 手順
1. `search_companies` で EDINETコードを取得
2. `get_company` で PER/PBR/配当利回り/EPS/BPS を取得
3. `get_financials` で過去5年の EPS 推移を取得（years=5）
4. `get_ranking` で業種内の PER / ROE / 配当利回りランキングを取得
5. `screen_companies` で同業種企業の平均的バリュエーションを確認

## 出力

### 現在のバリュエーション指標
| 指標 | 現在値 | 業種平均 | 乖離 |
|------|--------|---------|------|
| PER | | | 割安/適正/割高 |
| PBR | | | |
| 配当利回り | | | |
| EV/EBITDA | | | |

### EPSトレンド（過去5年）
- EPS の推移と CAGR
- 現在の PER が「割安」か「割高」かの根拠

### DDM（配当割引モデル）簡易試算
- 直近配当 / 過去3年平均配当成長率 / 想定割引率(8%) で理論株価を試算
- 現在株価との乖離率

### PBR解体分析
- ROE × PER = PBR の確認
- ROE が低い場合は「低ROE銘柄のバリュートラップリスク」を指摘

### 総合バリュエーション判定
- **強い買い** / **買い** / **中立** / **割高注意** / **売り検討**
- 判定根拠を3行で

出力は日本語・数値根拠を必ず示すこと。
