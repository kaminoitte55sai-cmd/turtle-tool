# 直近決算速報チェック
# 対象銘柄: {{TARGET_CODE}}

EdinetDB を使って {{TARGET_CODE}} の直近決算を分析してください。

## 手順
1. `search_companies` で EDINETコードを取得
2. `get_company` で基本情報・直近決算(latestEarnings)を取得
3. `get_earnings` で過去8四半期の決算推移を取得（limit=8）

## 出力

### 直近決算サマリー
- 決算期・開示日
- 売上高 / 営業利益 / 純利益 / EPS（前年同期比付き）
- 会社予想との乖離（上振れ/下振れ/概ね一致）

### 業績トレンド（直近8四半期）
- 売上・営業利益の推移を数値表で表示
- トレンドの方向性：加速↑↑ / 改善↑ / 横ばい→ / 悪化↓ / 急落↓↓

### 注目ポイント
- 良かった点（2〜3行）
- 懸念点（2〜3行）

### 判断
次回決算までの間、**買い継続 / 様子見 / 要注意** のいずれかを根拠付きで。

出力は日本語・簡潔に。
