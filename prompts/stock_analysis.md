# 銘柄総合分析プロンプト
# 使い方: claude -p "$(cat prompts/stock_analysis.md)" --allowedTools edinetdb

## タスク
EdinetDB MCP (`edinetdb`) を使って、指定銘柄の総合分析レポートを作成してください。

## 分析対象
- **銘柄コード**: {{TARGET_CODE}}  ← 実行前に証券コードに書き換えてください

## 手順

### Step 1: 基本情報の取得
- `get_company` で企業概要・最新財務・財務健全性スコアを取得

### Step 2: 財務トレンド分析
- `get_financials` で過去5年の時系列データを取得
- 売上・営業利益・EPS・ROEの推移を確認

### Step 3: 決算データの確認
- `get_earnings` で直近8四半期の決算短信データを取得
- 前年同期比の変化トレンドを把握

### Step 4: 大株主・株主構成の確認
- `get_shareholders` で主要株主リストを取得
- 機関投資家・外国人持株比率・安定株主比率を確認

### Step 5: 定性情報の分析
- `get_text_blocks` で有価証券報告書の以下セクションを取得:
  - 事業の状況（競合優位性・市場環境）
  - 事業等のリスク
  - 経営者による財政状態・経営成績の分析

### Step 6: AI分析の取得
- `get_analysis` で財務健全性スコア・業種ベンチマーク・AI所見を取得

## 出力フォーマット

### 1. 企業概要（3〜5行）

### 2. 財務ハイライト
- 直近本決算の主要数値（売上・営業利益・純利益・EPS）
- 前期比成長率

### 3. 5年間トレンド
- 売上・利益・ROEの方向性（拡大/横ばい/縮小）をグラフ代わりに記号で表現:
  `↑ 拡大基調` / `→ 横ばい` / `↓ 縮小傾向`

### 4. 株主構成の特徴
- 筆頭株主・外国人比率・機関投資家比率

### 5. 強みと課題
- 強み: 箇条書き3点
- 課題・リスク: 箇条書き3点

### 6. バリュエーション評価
- PER・PBR・配当利回りを業種平均と比較
- 割安/適正/割高の判定と根拠

### 7. 総合スコアと結論
- 財務健全性スコア（EdinetDB）: X/100
- 投資スタンス: 強気 / 中立 / 弱気
- 1〜2行の結論

## 出力言語: 日本語
